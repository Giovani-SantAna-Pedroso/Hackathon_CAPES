# Instalação

Execute o comando:

```bash
npm install --force
```

Crie o arquivo .env como o exemplo abaixo:

```
GEMINI_API_KEY="Sua_Api_key"
SERP_API_KEY="Sua_Api_key_Google_Search"
```
